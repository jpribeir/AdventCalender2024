To initialize repo
> git init
> it add <files>
> git commit -m "Comment here"
> git branch -M main
> git remote add origin https://github.com/USERNAME/REPOSITORY.git
> git push -u origin main

To clone and manage repo in new machine:
> git clone https://github.com/USERNAME/REPOSITORY.git

To submit new changes (branch is usually main):
> git add <files>
> git commit -m "Comment here"
> git push origin <branch>

To unstage file:
> git reset HEAD <file>

To get latest updates from repo
> git pull

To reset file from main
> git checkout origin [filename]
